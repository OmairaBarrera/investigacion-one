PROPIEDADES DE BACGROUNDS CSS

Background-color

Especifica color de fondo de un elemento

Opacity

Transparencia de un elemento, sus elementos secundarios heredan la opacidad.
Toma valores entre 0.0 y 1.0

Bienvenidos

Transparencia usando RGBA

Establece opacidad solo para el color de fondo

Bienvenidos

Background-image

Imagen para fondo de un elemento, la imagen se repite

Background-repeat

Indica como se debe repetir la imagen de fondo

  • Repeat-y
  • Repeat-x
  • no-epeat

Background-position

Indica la posiscion de la imagen de fondo

  • right top
  • right bottom
  • let top
  • let bottom

Background-attachmet

Imagen de fondo pueda desplazarce o no con el contenido

  • Fixed

    • se mueve con el contenido de la pagina, se observa el ejemple en el body

  • Scroll

Background

Especifica todas las propiedades de fondo en una sola propiedad
