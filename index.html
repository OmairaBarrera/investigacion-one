PROPIEDADES DE COLORS CSS

Los colores se especifican utilizando nombres de color predefinidos o valores RGB, HEX, HSL, RGBA, HSLA.

Puede establecer el color de fondo para los elementos, color del texto, color de los bordes

RGB

En CSS, un color se puede especificar como un valor RGB, utilizando esta fórmula: rgb ( rojo, verde , azul ); Cada parámetro (rojo, verde y azul) define la intensidad del color entre 0 y 255.

Tomato
rgb(255, 99, 71)

Orange
rgb(255, 165, 0)

Gray
rgb(128, 128, 128)

Los tonos de gris a menudo se definen utilizando valores iguales para las 3 fuentes de luz:

rgb(60, 60, 60)

rgb(120, 120, 120)

rgb(210, 210, 210)

RGBA

Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color. Un valor de color RGBA se especifica con: rgba ( rojo, verde , azul, alfa ) El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (nada transparente):

rgb(255, 0, 0,0.8)

rgb(255, 0, 0, 0.4)

rgb(255, 0, 0, 0)

HEX

Un color hexadecimal se especifica con: #RRGGBB, donde los enteros hexadecimales RR (rojo), GG (verde) y BB (azul) especifican los componentes del color.

Tomato
#ff6347

Orange
#ffa500

Gray
#808080

Los tonos de gris a menudo se definen utilizando valores iguales para las 3 fuentes de luz:

#3c3c3c

#787878

#d2d2d2

HSL

HSL significa tono, saturación y luminosidad.
En CSS, un color se puede especificar utilizando el tono, la saturación y la luminosidad (HSL) en la forma: hsl( tono , saturación , luminosidad ) El tono es un grado en la rueda de colores de 0 a 360. 0 es rojo, 120 es verde y 240 es azul. La saturación es un valor porcentual. 0% significa un tono de gris y 100% es el color completo. La ligereza también es un porcentaje. 0% es negro, 50% no es ni claro ni oscuro, 100% es blanco

Tomato
hsl(9, 100%, 64%)

Orange
hsl(39, 100%, 50%)

Gray
hsl(0, 0%, 50%)

Los tonos de gris a menudo se definen estableciendo el tono y la saturación en 0, y ajustando la luminosidad de 0 % a 100 % para obtener tonos más oscuros/claros:

hsl(0, 0%, 24%)

hsl(0, 0%, 47%)

hsl(0, 0%, 82%)
